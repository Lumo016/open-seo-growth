# Product Assets

This project includes a small starter asset set:

- `static/assets/logo.svg`
- `static/assets/social-preview.svg`
- `docs/assets/open-seo-growth-concept.png`

The design direction is an original open-source SaaS workbench:

- true light canvas, not a dark-only developer dashboard
- black and green primary action system
- compact analysis panels with 8px or smaller radius
- first screen contains the actual URL audit workflow
- beginner onboarding is treated as product functionality, not help text

The generated concept image is archived for design continuity. It is not required at runtime and should not be presented as an actual product screenshot.
